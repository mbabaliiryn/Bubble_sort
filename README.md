### Run tests

- The code has been tested using the following test cases:

* `bubble_sort(35, 15, 61, 453, 84)`
* `bubble_sort_by(["hi","hello","hey"]) {|i, j| i.length - j.length}`
* `bubble_sort(18, 24, 2, 93, 16)`
